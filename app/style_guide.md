# Style Guide

## Overview

This style guide provides the guidelines to ensure a consistent and user-friendly UI/UX across all parts of our application. Adhering to these guidelines will help maintain a cohesive design that is intuitive and visually appealing to our users.

## Color Scheme

### Primary Colors

- **Primary (Brand) Color**: `#007bff`
  - This color should be used for elements such as primary buttons, active links, and prominent UI components.
  
- **Secondary Color**: `#6c757d`
  - Use this color for less prominent elements such as borders, separators, and text that needs to stand out slightly from the background.

### Neutral Colors

- **Background (Light)**: `#f8f9fa`
  - This light gray should be used for the main background color of most pages.
  
- **Background (Dark)**: `#343a40`
  - Use this darker shade for section dividers, sidebar backgrounds, and other less prominent areas.

### Accent Colors

- **Success**: `#28a745`
- **Info**: `#17a2b8`
- **Warning**: `#ffc107`
- **Danger**: `#dc3545`

## Typography

- **Font Family**: Open Sans, sans-serif
  - This font provides a clean and modern look that is easy to read across different devices.

- **Header Font (Bold)**: Roboto Bold
  - Used for headings and titles to make them stand out more prominently.

### Sizes

- **Heading 1**: 2.5rem (40px)
- **Heading 2**: 2rem (32px)
- **Heading 3**: 1.75rem (28px)
- **Heading 4**: 1.5rem (24px)
- **Body Copy**: 1rem (16px)

### Line Height

- **Line Height for Body Copy**: 1.5

## Component Styles

### Buttons

- **Primary Button**:
  - Background color: `#007bff`
  - Text color: White
  - Hover state: Darken the background slightly.
  
- **Secondary Button**:
  - Background color: `#6c757d`
  - Text color: White
  - Hover state: Lighten the background slightly.

### Input Fields

- **Background Color**: `#fff`
- **Border**: `1px solid #ced4da`
- **Text Color**: `#212529`
- **Hover State**: Highlight border in `#80bdff`.

### Links

- **Normal**: Blue (`#007bff`)
- **Visited**: Purple (`#6c757d`)
- **Hover**: Darker blue (`#0056b3`)

### Cards

- **Background Color**: `#fff`
- **Border Radius**: 4px
- **Box Shadow**: `0 .25rem .75rem rgba(0,0,0,.15)`

## Grid and Layout

- **Grid System**: Use a responsive grid system to ensure consistency across different screen sizes.
- **Padding**: Consistent padding of `1rem` (16px) for standard spacing.

---

By following this style guide, we can maintain a high level of visual consistency and user experience in our application.
